Deploy AWS lambda function with Travis CI.

## requirements

- [NodeJS v12.18.3+](https://nodejs.org/en/) or [nvm v0.37.0+](https://github.com/nvm-sh/nvm/releases/tag/v0.37.0)
- [AWS Account](https://aws.amazon.com/)
  - Access Key Id (key)
  - Secret Access Key (secret)
